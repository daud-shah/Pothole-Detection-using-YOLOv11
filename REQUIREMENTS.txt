HOW TO INSTALL REQUIREMENTS
================================================================================

Option 1: Install Main Packages Only (Recommended)
-----------
pip install ultralytics opencv-python pillow

Option 2: Install from Requirements File (If You Create One)
-----------
pip install -r requirements.txt

Option 3: Install with Specific Versions
-----------
pip install ultralytics==8.4.41 opencv-python==4.13.0.92 pillow==12.2.0

================================================================================
SYSTEM SPECIFICATIONS
================================================================================

Python Version:
  - Required: Python 3.8 or higher
  - Recommended: Python 3.14 (current environment)
  - Tested on: Python 3.14.3

Operating System:
  - Windows 10/11 (Primary development environment)
  - Linux/Mac (Should work with modifications)

RAM Requirement:
  - Minimum: 4GB
  - Recommended: 8GB or higher

GPU (Optional):
  - NVIDIA GPU with CUDA support (recommended for faster detection)
  - CPU-only mode works but slower
  - PyTorch will auto-detect available GPU

Disk Space:
  - Virtual environment: ~2-3GB
  - YOLO11m model weights: ~50-100MB (downloaded on first use)
  - Total project: ~3-4GB

================================================================================
CREATING A REQUIREMENTS.TXT FILE
================================================================================

To create a requirements.txt file for your project, run:

pip freeze > requirements.txt

This will save all installed packages to requirements.txt.

Future users can then install all requirements with:

pip install -r requirements.txt

================================================================================
TROUBLESHOOTING INSTALLATION
================================================================================

Issue: "pip is not recognized"
Solution: Use: python -m pip install <package>

Issue: Permission denied
Solution: Try: pip install --user <package>

Issue: Long installation time
Solution: Large packages like torch and torchvision take time. Be patient.

Issue: Build errors
Solution: Ensure you have Visual C++ build tools installed on Windows
